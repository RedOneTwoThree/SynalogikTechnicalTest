To run the program run the following commands

1) In the root folder run "mvn clean install" this will run the test and create the jar file which will be put into the target folder

2) To run the program first run the following 
"java -cp target/count-1.0-SNAPSHOT.jar Count"

3) After that all you need to do is enter the directory of the txt file. The root directory is the count folder. A copy of the bible.txt is available there.
